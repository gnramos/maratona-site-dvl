A 1ª Fase e a Final Brasileira da Maratona de Programação têm correção centralizada, portanto o ambiente computacional é o mesmo independentemente da sede.

O sistema de submissão eletrônica utilizado é o BOCA. O sistema de correção das submissões será executado utilizando o ambiente computacional Maratona Linux, que é implantado nas sedes pelo boot do pendrive disponibilizado pela organização. O responsável pelos sistemas usados na Maratona de Programação é o Prof. Bruno Ribas (UnB - Gama).
